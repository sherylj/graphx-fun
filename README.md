
graphx-fun
==========

graphx sandbox for graph load, construction and operations


**Sofware Install:**

 - Spark (1.0.0)
 - Scala (2.10.4)
 - SBT (0.13.5)

**Project Setup**

        git clone <project url url>
        sbt package
        sbt run


**Data/Resources**

Graph data taken from http://snap.stanford.edu/data/
